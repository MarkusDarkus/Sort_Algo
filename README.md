## Sorting

### Insrtion Sort
Сортировка вставками. Строит отсортированную последовательность, путем постепенного включения элементов из исходного массива.

* Лучший случай - \Omega(n)
* Худший случай - O(n^2)
* Средний случай - O(n^2)
* Память - O(1)
* Стабильность - YES
